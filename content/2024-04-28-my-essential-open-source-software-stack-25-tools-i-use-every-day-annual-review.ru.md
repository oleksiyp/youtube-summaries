# Обзор Open Source программного обеспечения

## Вступление

Ежегодный обзор open source инструментов, которые автор регулярно использует. Видео охватывает множество проектов с краткими описаниями и ссылками на подробные материалы.

---

## Инструменты управления серверами

### Webmin
- **Назначение**: Графический интерфейс для управления Linux-серверами
- **Преимущества**: 
  - Простая установка через стандартные репозитории
  - Альтернатива командной строке для администрирования
  - Мониторинг CPU, памяти, дисков
  - Управление файловыми системами, безопасностью, логами
  - Возможность управления несколькими серверами из единого интерфейса
- **Целевая аудитория**: Пользователи, переходящие с Windows Server или желающие иметь GUI вместо CLI

### Pi-KVM
- **Описание**: Аппаратно-программное решение на базе Raspberry Pi
- **Функции**: KVM-переключатель для удаленного доступа к физическим машинам
- **Применение**: Идеален для MSP (Managed Service Providers) и удаленного управления клиентскими системами

## Сетевые инструменты и обратные прокси

### Nginx Proxy Manager
- **Функции**: Простое управление обратными прокси-серверами
- **Возможности**:
  - Графический интерфейс для настройки
  - Автоматические SSL-сертификаты
  - Списки контроля доступа
  - Фильтрация по IP-адресам
  - Аутентификация для защищенного доступа

### NetBird
- **Тип**: VPN-решение на базе WireGuard
- **Особенности**:
  - Простая настройка VPN-соединений
  - Управление пользователями и группами
  - Гранулированный контроль доступа к машинам
  - Мониторинг состояния подключений (онлайн/оффлайн)
- **Применение**: Безопасные соединения для MSP и удаленной работы

## Инструменты удаленного управления

### MeshCentral
- **Описание**: Полноценная RMM (Remote Monitoring and Management) система
- **Возможности**:
  - Удаленный рабочий стол через браузер
  - Доступ к терминалу/командной строке
  - Управление файлами (загрузка/скачивание)
  - Детальное логирование действий
  - Расширенный набор инструментов администрирования
- **Недостаток**: Пока нет поддержки Wayland
- **Планы**: Будет использован в серии "Создание MSP на Open Source"

### Remotely
- **Функции**: Инструмент удаленного рабочего стола и поддержки
- **Особенности**:
  - Двухфакторная аутентификация
  - Клиенты для Windows и Linux
  - Доступ по коду через браузер
  - Не требует регистрации для конечных пользователей
- **Статус**: Активно развивается, компания сохранила open source лицензию

### Guacamole
- **Описание**: Clientless удаленный рабочий стол через браузер
- **Поддержка протоколов**: RDP, VNC и другие
- **Возможности**:
  - Группировка подключений
  - Доступ к локальным машинам через единый интерфейс
  - Полноэкранный режим
  - Не требует установки клиентского ПО

## Контейнеризация и виртуализация

### Portainer
- **Назначение**: GUI для управления Docker-контейнерами
- **Преимущества**:
  - Визуальный обзор контейнеров и их статусов
  - Управление контейнерами одним кликом
  - Просмотр логов
  - Доступ к shell контейнеров
- **Версии**: Community Edition (бесплатная) и Business Edition

### LXD Console (с Incus)
- **Функции**: Веб-интерфейс для управления LXD/Incus контейнерами
- **Возможности**:
  - Мониторинг ресурсов (CPU, память)
  - Управление контейнерами и виртуальными машинами
  - Просмотр IP-адресов и конфигураций
  - Встроенная веб-консоль для доступа к контейнерам
  - Управление несколькими серверами

## Безопасность и аутентификация

### Vaultwarden
- **Описание**: Легковесный форк Bitwarden на Rust
- **Функции**:
  - Менеджер паролей с шифрованием
  - Двухфакторная аутентификация
  - Синхронизация с клиентами Bitwarden
  - Интеграция с браузерами
  - Семейный доступ и совместное использование
- **Преимущества**: Меньше ресурсоемкий, чем оригинальный Bitwarden

### Authentik
- **Описание**: Решение для единого входа (SSO)
- **Поддерживаемые протоколы**: LDAP, SAML, OIDC
- **Применение**:
  - Централизованная аутентификация для всех сервисов
  - Портал доступа к приложениям
  - Управление пользователями и правами доступа
- **Интеграция**: Используется с MeshCentral, Nextcloud, Headscale, NetBird и другими

## Аналитика и мониторинг

### Metabase
- **Назначение**: Инструмент для создания дашбордов и отчетов
- **Возможности**:
  - Визуализация данных с помощью графиков
  - Отслеживание бизнес-метрик
  - Анализ трендов
  - Интуитивный интерфейс с минимальным кодированием
- **Пример использования**: Отслеживание продаж на eBay, анализ сезонности, учет расходов на доставку
- **Версии**: Бесплатная и профессиональные версии для бизнеса

## Облачные хранилища и коллаборация

### Nextcloud
- **Описание**: Полнофункциональная платформа для работы с файлами и коллаборации
- **Основные функции**:
  - Хранение и синхронизация файлов
  - Календари и контакты
  - Управление задачами (To-Do списки)
  - Расширяемость через приложения
- **Настройка**: Nextcloud All-in-One (AIO) для простой установки
- **Интеграция**: Работает с Authentik для SSO

## Коммуникации

### Rocket.Chat
- **Назначение**: Платформа для командного общения
- **Применение**: Сообщество для подписчиков канала
- **Функции**:
  - Каналы для обсуждений

### Matrix (Dendrite)

- **Применение**: Семейный чат
- **Преимущества**:
  - Приватные коммуникации
  - Самостоятельный хостинг
  - Клиенты для desktop и mobile
  - Групповые чаты и личные сообщения

## Ведение заметок и документации

### BookStack
- **Назначение**: Организация документации и инструкций
- **Структура**: Полки → Книги → Главы → Страницы
- **Особенности**:
  - Мощный поиск
  - Пошаговые инструкции для всех видео
  - Интеграция с Authentik
  - Markdown-поддержка

### Trilium Notes
- **Описание**: Иерархическое приложение для заметок
- **Функции**:
  - Быстрый поиск
  - Древовидная структура
  - Собственный сервер синхронизации
  - Electron-клиент для desktop
- **Применение**: Личные списки (например, список фильмов для семейных просмотров)
- **Альтернатива**: Obsidian, Logseq, Joplin

## Контент-менеджмент

### WriteFreely
- **Назначение**: Минималистичная платформа для блогинга
- **Особенности**:
  - Простая настройка
  - Публичные или приватные блоги
  - Поддержка изображений
- **Пример использования**: Проект реставрации пианино с фотодневником

### Pixelfed (упомянут как Pixer)
- **Описание**: Самостоятельный хостинг изображений
- **Функции**:
  - Загрузка и организация фото
  - Генерация URL для вставки в посты
  - Интеграция с блог-платформами
- **Применение**: Хостинг изображений для WriteFreely

## Домашняя автоматизация

### Home Assistant
- **Описание**: Мощная платформа домашней автоматизации
- **Преимущества**:
  - Полная локальная работа без облака
  - Конфиденциальность данных
  - Интеграция с множеством устройств
  - Автоматизации без необходимости писать код
- **Эволюция**: Раньше требовались YAML-файлы, теперь всё через GUI
- **Функции**:
  - Управление камерами
  - Дашборды для мониторинга
  - Сложные сценарии автоматизации

## Создание контента

### Shotcut
- **Назначение**: Видеоредактор
- **Преимущества над альтернативами**:
  - Быстрый и эффективный рабочий процесс
  - Интуитивная временная шкала
  - Система клипов
  - Настраиваемые виды и метры
  - Меньшая кривая обучения чем у DaVinci Resolve или Adobe
- **Развитие**: Постоянные обновления с новыми функциями

### OBS (Open Broadcaster Software)
- **Назначение**: Запись экрана и стриминг
- **Преимущество**: Поддержка Wayland
- **Причина перехода**: Предыдущий инструмент (Simple Screen Recorder) не поддерживает Wayland
- **Актуальность**: Wayland становится стандартом, X11 постепенно уходит

---

## Dashy
- **Описание**: Настраиваемый дашборд для быстрого доступа к сервисам
- **Функции**:
  - Визуальная организация ссылок на сервисы
  - Проверка доступности сервисов (ping)
  - Визуальный редактор (не только YAML)
  - Доступ к домашним и облачным сервисам
- **Преимущество**: Простота настройки и использования
